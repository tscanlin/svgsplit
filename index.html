What is this thing?

SVGsplit is made for splitting "groups" (<g> tags) in svg files down into seperate svg files. It then outputs them on the page and automatically downloads them, packaged up as a zip file. This is useful for creating SVG icons and works quite well with Adobe Illustrator or Inkscape. Just save your source file as an SVG and drop it in the box above. All the layer names will become the new file names for the split up SVGs. Email me if you have any questions: tim[at]tscanlin.com
